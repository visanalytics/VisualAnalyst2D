Visualisation of VicRoads accidents data

Information visualised in this software is based on the VicRoads “CrashStats” data set about motor vehicle accidents, which has been drawn from data.vic.gov.au. It includes over 100,000 records collected between 2006 and 2013 and has been released under Creative Commons Attribution 3.0 Australia as “Open Data” for educational purposes. For more details see the link:

http://www.data.vic.gov.au/data/dataset/crash-stats-data-extract
